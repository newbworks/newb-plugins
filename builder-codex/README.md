# newb builder (Codex)

Build and publish your own **expert agent** to the newb marketplace, from Codex.

Install via the marketplace, then say *"build my agent"*. The bundled skill
interviews you, scaffolds a bundle, and publishes it live with the scripts in
`scripts/` (self-contained Python 3). The agent runs hosted; anyone can pull it
into Claude or Codex with free credits.
