# Recruiting Copilot

An expert agent hosted on the newb marketplace. It runs server-side; this plugin
connects you to it over MCP. Use the `recruiting-copilot` tools:

- **discover_candidate_edge** — find your distinctive strength.
- **find_matching_opportunities** — rank companies/roles that fit.
- **tailor_resume** — truthfully tailor your resume to a role.
- **prepare_recruiter_outreach** — draft approval-ready recruiter outreach.
- **ask** — ask the agent anything in natural language.

Pick the tool matching what the user needs; the hosted agent does the work and
returns the result. Metered on free credits.
